# 键盘快捷键

## 默认快捷键方案

Rectangle 提供两种默认快捷键方案：

### 推荐方案（alternateDefaultShortcuts = true）

| 操作 | 快捷键 |
|------|--------|
| 左半屏 | Ctrl + Option + ← |
| 右半屏 | Ctrl + Option + → |
| 上半屏 | Ctrl + Option + ↑ |
| 下半屏 | Ctrl + Option + ↓ |
| 左上角 | Ctrl + Option + U |
| 右上角 | Ctrl + Option + I |
| 左下角 | Ctrl + Option + J |
| 右下角 | Ctrl + Option + K |
| 最大化 | Ctrl + Option + Return |
| 第一三分之一 | Ctrl + Option + D |
| 前三分之二 | Ctrl + Option + E |
| 中间三分之一 | Ctrl + Option + F |
| 后三分之二 | Ctrl + Option + T |
| 最后三分之一 | Ctrl + Option + G |
| 放大 | Ctrl + Option + = |
| 缩小 | Ctrl + Option + - |
| 居中 | Ctrl + Option + C |
| 恢复 | Ctrl + Option + Delete |
| 上一显示器 | Ctrl + Option + Cmd + ← |
| 下一显示器 | Ctrl + Option + Cmd + → |
| 最大化高度 | Ctrl + Option + Shift + ↑ |

### Spectacle 方案（alternateDefaultShortcuts = false）

| 操作 | 快捷键 |
|------|--------|
| 左半屏 | Cmd + Option + ← |
| 右半屏 | Cmd + Option + → |
| 上半屏 | Cmd + Option + ↑ |
| 下半屏 | Cmd + Option + ↓ |
| 左上角 | Ctrl + Cmd + ← |
| 右上角 | Ctrl + Cmd + → |
| 左下角 | Ctrl + Shift + Cmd + ← |
| 右下角 | Ctrl + Shift + Cmd + → |
| 最大化 | Cmd + Option + F |
| 放大 | Ctrl + Option + Shift + → |
| 缩小 | Ctrl + Option + Shift + ← |
| 居中 | Option + Cmd + C |
| 恢复 | Ctrl + Option + Delete |
| 上一显示器 | Ctrl + Option + Cmd + ← |
| 下一显示器 | Ctrl + Option + Cmd + → |
| 最大化高度 | Ctrl + Option + Shift + ↑ |

## 恢复默认快捷键

在偏好设置的 Shortcuts 标签页中点击「Restore Default Shortcuts」可恢复默认快捷键。

## 自定义快捷键

- 所有窗口操作均可在偏好设置中自定义快捷键
- 启用「Allow any shortcut」后可绑定任意快捷键组合
- 使用 [Key Codes](https://apps.apple.com/us/app/key-codes/id414568915) 等应用可查看键码和修饰键整数值

## 修饰键整数值（用于终端配置）

| 修饰键 | 整数值 |
|--------|--------|
| cmd | 1048576 |
| option | 524288 |
| ctrl | 262144 |
| shift | 131072 |
| fn | 8388608 |

组合时相加，如 Cmd + Option = 1572864。
